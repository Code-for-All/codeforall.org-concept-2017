Join Us

There are three ways to participate:

  1. Access our online resources and toolkits.
  2. Participate in the Code for America Brigade network. The Code for America Brigade network is open to members from outside the US. International Brigade chapters can join as Brigade Network Members. More information and a sign up form can be found on the Brigade website).
  3. Become an institutional partner member of Code for All. If you are an established institution in your home country and want to join Code for All as an official partner organization, send an email with more information about your work to international [at] codeforamerica.org. If this level of engagement is of interest to you, but you're at the front end of your organizing journey, starting a Brigade is a great first step towards joining the Code for All network (see option 2).

Team Contact: international [at] codeforamerica [dot] org .
