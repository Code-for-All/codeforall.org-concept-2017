Code for Australia

Our mission is to improve the interface between citizens and government via civic technology, and in return create a civic-tech ecosystem in Australia.

codeforaustralia.org

Program Description

Code for Australia is a non-profit organisation working with citizens, industry and government to develop new ways to solve civic problems. We build open source technology and organize a network of people dedicated to making government services simple, effective, and easy to use.
The organisation has three programs:
- Fellowship
- Incubator
- Civic Lab
Areas of interest: diversity in technology sector, demystification of technology, collaboration across government, industry and citizens, and design. Code for Australia works across local and state government.

Founding Story

Alvaro Maz met Dan Groch in 2013 doing a startup leadership course in Melbourne. Whilst both were building their business, Dan had also started thinking about contextualising the Code for America model to Australia with Jacob Lindsay. The motivations of the group include doing things differently, making change, demonstrating things can happen and helping others succeed.
